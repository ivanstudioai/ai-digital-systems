AI Insight Generator System

This project demonstrates an applied AI system designed to help generate
and clarify meaningful insights from loosely defined prompts or questions.

The system starts with a broad or ambiguous idea and produces:
- a focused insight that reframes the topic
- a short explanation that adds context and meaning

Input:
- A free-form prompt describing a theme, question, or area of curiosity

Outputs:
- A concise insight that captures a core idea
- An explanation that unpacks why the insight matters

Structure:
- input/prompt.txt
- output/insight.txt
- output/explanation.txt

This type of system reflects a common real-world need in creative,
strategic, and business contexts: turning vague thoughts into clear,
usable perspectives.

The focus of this project is framing, clarity of thought, and repeatable
idea generation rather than optimization or automation.
