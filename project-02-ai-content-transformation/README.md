AI Content Transformation System

This project demonstrates an applied AI system that transforms a single
source text into multiple usable content formats.

The goal is to show how structured prompting and AI tools can be used
to efficiently repurpose ideas across different communication channels.

Input:
- A short source text containing a core idea or explanation

Outputs:
- A concise written summary for quick understanding
- A short audio explainer for spoken communication
- A social-ready post suitable for professional platforms

Structure:
- input/source-text.txt
- output/summary.txt
- output/social-post.txt
- output/audio.mp3

This system reflects a common real-world need: turning one piece of
knowledge into multiple formats without rewriting from scratch.

The focus of this project is clarity, reproducibility, and practical
system design rather than content performance or optimization.
