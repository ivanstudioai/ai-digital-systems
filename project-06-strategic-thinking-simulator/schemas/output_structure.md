# Strategic Thinking Simulator — Output Schema (V1)

## For Each Perspective Agent

### 1. Hidden Assumptions
- List implicit assumptions the idea relies on.

### 2. Failure Scenarios
- Describe plausible ways the idea could fail.

### 3. Dependency Risks
- External systems, partners, timing, infrastructure dependencies.

### 4. Edge Cases
- Situations where the idea breaks, behaves unexpectedly, or creates risk.

### 5. Opposing Arguments
- Strong counter-position against the idea from this perspective.
